# vue-batch-export

> A Vue plugin to batch export
> 纯前端实现对表格数据的批量导出成xls文件，支持超过1000条以上数据

## Document

### argument

| Param | Type | Describe |
| :------: | :------: | :------: |
| url | String | 导出表格的请求路径 |
| params | Object | 导出表格的请求参数 |
| export_type | String | 导出表格类型 |
| total | Number | 需要导出数据的总数量 |
